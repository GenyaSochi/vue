При использовании Options API логика компонентов определяется с помощью объекта опций, таких как data, methods и mounted Свойства, определённые в опциях, доступны через this. Options API сосредоточен вокруг концепции «экземпляра компонента» Composition API сосредоточен на объявлении переменных реактивного состояния. Метод export default в Vue используется для экспорта компонента из файла. Это позволяет повторно использовать компонент в других частях приложения. Чтобы использовать метод export default, нужно создать новый файл в проекте с именем MyComponent.vue. В этом файле определить разметку и логику компонента. Экземпляр приложения не будет ничего отрисовывать, пока не вызвать его метод .mount() app.mount('#app') Директива v-bind сообщает Vue поддерживать атрибут id элемента в синхронизации со свойством dynamicId компонента. Если связанное значение будет null или undefined, то атрибут будет удален из отрисованного элемента. Если перед атрибутом поставить : то он станет динамическим и в нем можнгиспользовать js.
